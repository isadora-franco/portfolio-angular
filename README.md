# Portfólio — Isadora Franco

Projeto de portfólio desenvolvido com Angular 17.

## Como rodar

```bash
npm install
npm start
```

Acesse em: http://localhost:4200

## ⚠️ Importante — CV em PDF

Adicione o arquivo do seu currículo em PDF em:
`src/assets/pdf/cv.pdf`

O botão "Download CV" na página vai apontar para esse arquivo.
